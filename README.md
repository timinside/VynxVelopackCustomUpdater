# VynxVelopackCustomUpdater
Custom updater for games that uses Velopack (example: "osu!"). This updater sending commands to Vynx Games Launcher to install game updates.

## Another example:
"osu!" game is downloading update and to install uses Update.exe by Velopack, but we replace Update.exe to our version for not using Velopack installation system.

### Updater is powered by TCC and it's for Windows Only! You can compile it by using "build.bat". Before compiling .exe replace original icon (icon.ico) with your icon.
